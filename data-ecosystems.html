Data Commons

The paper below is a short commentary summarizing ten lessons learned over the past decade building data commons. In particular, it emphasizes the importantance of curating the data in a commons using one (or more data models) and harmonizing the data using a common set of bioinformatics pipelines.

  • Grossman, Robert L. Ten lessons for data sharing with a data commons. Scientific Data 10, no. 1 (2023): 120. pdf
